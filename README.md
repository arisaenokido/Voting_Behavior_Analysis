#Voting Behavior Analysis (Canada) 
## Overview
This project analyzes Canadian federal election data to examine the relationship between median income and voting patterns across electoral districts.

Using R, the project applies data cleaning, exploratory data analysis (EDA), visualization, and regression modeling to identify socioeconomic trends in political behavior.

## Objectives
- Investigate how income levels relate to voting outcomes across political parties  
- Clean and transform real-world election data for analysis  
- Visualize relationships between income and vote share  
- Apply statistical models to quantify correlations

## Dataset
The dataset includes Canadian federal election results by riding, along with socioeconomic indicators such as median income.

Key variables:
- Median income
- Vote share by political party
- Electoral district (riding)

## Methodology

### Data Cleaning
- Converted variables to appropriate data types (numeric, factors)
- Handled missing and inconsistent values

### Exploratory Data Analysis
- Calculated summary statistics (mean, median, standard deviation)
- Identified trends across ridings

### Visualization
- Used ggplot2 to create scatter plots and trend lines
- Compared income levels with vote share across parties

### Statistical Analysis
- Applied linear regression models to analyze relationships between income and voting behavior
